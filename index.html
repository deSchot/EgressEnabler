The Egress Enabler

A browser-native rebuild of the Smedberg et al. (2023) egressibility instrument. Mark each environmental barrier compliant or non-compliant and watch the Egress Enabler Score recompute across all 14 functional limitations — then flip to Impact mode to see how your design changes move egressibility, à la Figure 3 of the paper.

Personal Component

The 14 functional limitations (A–M) the score is computed against. In the Egress Enabler these classify where egressibility issues fall rather than describing any one individual. Max = highest possible score per limitation.

Add a sub-scale

How scoring works. Each barrier item carries a 14-value scoring pattern (severity 0–4) — one value per functional limitation. When you mark an item non-compliant (the grey answer in the original), its pattern is added to the running score for each limitation. Sub-scales of the same type are weighted to sum to 1, so buildings of differing complexity stay comparable. Star rating reflects EES ÷ the published per-limitation maximum (0 issues = 4★; ≥75% of max = 0★).
